## Overview

This is a console-based Ecommerce application developed using Java and Object-Oriented Programming principles. The project simulates the core functionalities of an online shopping platform, allowing users to browse products, manage carts, place orders, and interact with a simple ecommerce workflow.

The project demonstrates the practical application of Java classes, objects, inheritance, encapsulation, and modular software design.

## Class Responsibilities

### Main.java
Entry point of the application and controls program execution.

### User.java
Handles user-related information and operations.

### Product.java
Represents products available in the store.

### Cart.java
Manages cart operations including adding and removing products.

### Order.java
Processes customer orders and maintains order information.

### Store.java
Acts as the central controller for managing products, users, and orders.

## Installation & Execution

### Clone the Repository

```bash
git clone https://github.com/srinidhi829/Ecommerce-java-project.git
```

### Navigate to Project Folder

```bash
cd Ecommerce-java-project
```

### Compile the Project

```bash
javac *.java
```

### Run the Application

```bash
java Main
```
