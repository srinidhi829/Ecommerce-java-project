# 🛒 Ecommerce Java Project

## 📌 Overview

This is a console-based Ecommerce application developed using Java.
It simulates basic online shopping functionalities like managing users, products, cart, and orders.

---

## 🚀 Features

- User management
- Product listing
- Add to cart functionality
- Order processing
- Store management

---

## 🛠️ Technologies Used

- Java
- OOP Concepts

---

## 📂 Project Structure

- `Main.java` → Entry point of the application
- `User.java` → Handles user details
- `Product.java` → Product information
- `Cart.java` → Cart operations
- `Order.java` → Order processing
- `Store.java` → Store management

---

## ▶️ How to Run

1. Compile all files:

   ```bash
   javac *.java
   ```

2. Run:

   ```bash
   java Main
   ```
